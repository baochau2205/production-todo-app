# Production Todo App

A frontend Todo application built with vanilla JavaScript using a production-style architecture.

## Features
- Add / delete / toggle tasks
- Filter (All / Active / Done)
- Persistent storage (localStorage)
- Event delegation pattern
- Clean separation of state / logic / UI

- HTML

## Architecture
The project is structured into:
- State layer (data management)
- Actions layer (business logic)
- Render layer (UI rendering)
